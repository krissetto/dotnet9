# Ascii Server (.NET 9)

A simple web server that returns ASCII art using Figgle.

## Prerequisites

- .NET 9.0 SDK
- Docker (optional)

## Running Locally

```bash
dotnet run
```

The server will be available at http://localhost:4032

## Development

This project uses:

- .NET 9.0
- Figgle (Figlet library for .NET)
- Minimal API pattern
